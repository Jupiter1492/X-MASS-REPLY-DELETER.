# X-MASS-REPLY-DELETER.
This Deletes X (Twitter) Replies you've made in the past. Use with Caution: It Deletes them all.

How to use it:
1. Go to your Profile page on X.
2. Click on the Replies tab.
3. Open the Developer Tools (F12 or Right Click -> Inspect).
4. Go to the Console tab, paste the code, and hit Enter.
5. Keep the tab open and active on your screen.

NOTE: Because X has to process actual data deletion on their end, rate limits for deleting tweets are sometimes slightly stricter than just unliking things. If it stalls, refresh the page and give it a short break before running it again.
